Search Engine Commands
============

_A list of commonly used Search Engine Commands_

--

### Core Commands

| Command | Description |
| ------- | ----------- |
| `ls` | Short Listing  |
| `ls -l` | Long Listing |
| `ls -a` | Listing incl. hidden files  |
| `cd` | Home directory |
| `cd [PATH]` | Change directory to path |
| `cd ..` | Change directory to parent |
